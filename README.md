# CS320-Software-Testing-Automation-and-Quality-Assurance

To ensure my program functions correctly, I would leverage automation to enhance both accuracy and efficiency. I would collaborate with a peer to review the code for any errors and conduct a final check myself once everything is complete.

To understand the user’s needs, I would strive to gain a clear grasp of the requirements and actively seek ongoing feedback to customize the program to better meet their needs.

In software design, I begin with thorough planning. I then develop the code in segments, employing an iterative approach to facilitate more efficient changes.
